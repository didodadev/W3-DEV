1. Mevcut bir projeden işlerin seçilmesiyle
2. Hazırlanan belgenin import edilmesiyle


 Belgeden Import Kuralları

Dosya uzantısı csv olmalı ve alan araları noktalı virgül (;) ile ayrılmalıdır. Format UTF-8 olmalıdır. Aktarım işlemi dosyanın 2. satırından itibaren başlar bu yüzden birinci satırda alan isimleri olmalıdır. Bu belgede olması gereken alan sayısı 17 tanedir. Alanlar sırasıyla ;

1 - Unique ID* : İşin Export edildiği programdaki ID. (Tam Sayı olarak değer girilmelidir) Zorunlu alan.
2 - İş Kategorisi* : Workcube iş kategori ID (Tam Sayı olarak değer girilmelidir) Zorunlu alan. Boş olan alanlar için default değer XML den atanabilir.
3 - Aşama* : Workcube iş aşama ID (Tam Sayı olarak değer girilmelidir) Zorunlu alan. Boş olan alanlar için default değer XML den atanabilir.
4 - Öncelik* : Workcube iş öncelik ID (Tam Sayı olarak değer girilmelidir) Zorunlu alan. Boş olan alanlar için default değer XML den atanabilir.
5 - Görevli* : Workcube çalışan ID (Tam Sayı olarak değer girilmelidir) Zorunlu alan. Boş olan alanlar için default değer XML den atanabilir.
6 - İş No : Atanacak işin bir numarası varsa kullanılabilir. (Girilecek karakter sayı max 40)
7 - Konu : İşin Başlığı (Girilecek karakter sayı max 150)
8 - PBS Kodu : Pbs Kodu (Girilecek karakter sayı max 50)
9 - Açıklama : İşin detayında kullanılmak için text formatta girilir.
10 - Öngörülen Süre : İşin tamamlanması için öngörülen süre Saat cinsinden olmalıdır. (Örnek : 2.5 )
11 - Süre : İşin tamamlanması için gerekli gün sayısı (Girilecek karakter sayı max 43)
12 - Başlama Tarihi : İşin başlama tarihi (GG/AA/YY)
13 - Bitiş Tarihi : İşin bitiş tarihi (GG/AA/YY)
14 - Tahmini Miktar : İşin kaç kere yapılacağı (Tam Sayı olarak değer girilmelidir)
15 - Milestone : İşin Milestone olup olmadığını belirtir. (Değer olarak 0 yada 1 girilmelidir)
16 - İş Gurubu : Workcube iş gurubu ID (Tam Sayı olarak değer girilmelidir) Zorunlu alan. Boş olan alanlar için default değer XML den atanabilir.
17 - Aktiflik Durumu : İşin aktiflik durumu (Değer olarak 0 yada 1 girilmelidir)
