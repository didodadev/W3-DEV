Journal Book (Day-today Book), It is the book that the commercial processes are listed in a chronological order, article by article, with the help of archives and company documents.

Starting and Ending Date: It will display the actions between the selected dates.

Account Type: Brings the Monotonous or UFDS codes and details of the cccount codes and account details sections.

Process Type: Will only bring the account receipt registries of the selected process types. If not selected, it will bring the activity reports of all receipts.

Report Type: Enables to extract reports in either Pdf or Excel format.

Page Layout: Enables extracting reports in A4 or Letter (continuous form) format.

Character Count-Account Title: This determines the visible character count of account titles. For instance, the character count of "320.01.001 – Workcube E-Business Co Inc." is 40. If you decrease the character count to 20 for reports, the title will show up as Workcube E-Business.

Character Count Details: This determines the extent of the visible parts of the details section. For instance, the character count for "Gasoline Expense of Counsellers in March" is 40. If you decrease the character count to 30 for reports, it will show up as "Gasoline Expense of Counseller".

Font and Font Size: Can be selected to determine the font type and font size.

Row Count: It determines how many rows will a page consist of. Character count and font type and size would cause disalignment in this case. In order to extract a smooth report, standards would be set in accordance with the default values in the report.

PLEASE NOTE: When you change the font size and character count, this will cause disalignments in the rows, that's why these properties would be altered accordingly, or never to be touched at all. For instance, if you wish to increase the font size, you should decrease character count or vice versa.
