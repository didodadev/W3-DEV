Projenize iş kopyalama işlemi 2 yöntemle yapılmaktadır.
1. Mevcut bir projeden işlerin seçilmesiyle
2. Hazırlanan belgenin import edilmesiyle


 Belgeden Import Kuralları

Dosya uzantısı csv olmalı ve alan araları noktalı virgül (;) ile ayrılmalıdır. Format UTF-8 olmalıdır. Aktarım işlemi dosyanın 2. satırından itibaren başlar bu yüzden birinci satırda alan isimleri olmalıdır. Bu belgede olması gereken alan sayısı 4 tanedir. Alanlar sırasıyla ;

1 - Ana İş ID / İş No* : Ana iş ID veya No (Tam Sayı olarak değer girilmelidir). Eğer ID ile giriş yapılıyorsa ana işe de ilişkilenen işe de ID yazılacak, iş no ile giriş yapılıyorsa iş no yazılacak. Zorunlu alan.
2 - İlişki Şekli* : Ana işin; ilişkilenen iş ile arasındaki ilişki. Finish to Start = 1, Start to Finish = 2, Start to Start = 3, Finish to Finish = 4. Zorunlu alan.
3 - İlişkilenen İş ID / İş No* : İlişkilenen iş ID veya No. Giriş yönteminde (ID ya da İş no seçimi) yapılan seçim burası için de geçerli olacak. Eğer ID ile giriş yapılıyorsa ana işe de ilişkilenen işe de ID yazılacak, iş no ile giriş yapılıyorsa iş no yazılacak. Zorunlu alan.
4 - Bekleme (Gün) : Bekleme süresi gün cinsinden tam sayı değil ise virgül ile ayrılacak.
